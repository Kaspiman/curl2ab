# Curl2ab

Convert your cURL command to [ab](http://httpd.apache.org/docs/current/en/programs/ab.html)

## Install

### Local

```
docker compose up
# or without docker
npm install && npm run start
```

Go to `localhost:1234`

### Production

```
docker compose run app npm run build
# or without docker
npm install && npm run build
```
